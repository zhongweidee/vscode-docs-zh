# 部署 Node.js Web 应用

Visual Studio Code 的 [Azure Tools](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-node-azure-pack) 扩展让部署 Node.js 应用程序变得简单。

![Azure Tools extension](images/azure/azure-tools.png)

## 部署教程

以下来自 Microsoft Learn 的教程介绍了通过 Visual Studio Code 将 Node.js 应用创建并部署到 Azure 的不同方法：

教程 | 说明 | 相关工具
--- | --- | ---
[部署 Azure Functions](https://learn.microsoft.com/azure/developer/javascript/tutorial/azure-function-cosmos-db-mongo-api) | 直接在 VS Code 中<br>构建和管理 Azure Functions 无服务器应用 | [Azure Resources](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azureresourcegroups) <br> [Azure Functions](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azurefunctions) <br> [Azure Databases](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-cosmosdb)
[使用 Azure 门户<br>和 Azure 应用服务进行部署](https://learn.microsoft.com/azure/app-service/tutorial-nodejs-mongodb-app) | 使用 Azure 应用服务<br>直接在 VS Code 中管理 Azure 资源 | [MongoDB](https://www.mongodb.com/docs/manual/installation/) <br> [Azure 门户](https://portal.azure.com/)
[使用 Docker 进行部署](https://learn.microsoft.com/azure/developer/javascript/tutorial/tutorial-vscode-docker-node/tutorial-vscode-docker-node-01) | 使用容器部署您的网站 | [容器工具](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-containers) <br> [Azure CLI](https://learn.microsoft.com/cli/azure/install-azure-cli) <br> [Azure 应用服务](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azureappservice) <br> [Azure Resources](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azureresourcegroups)
