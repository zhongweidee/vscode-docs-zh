# 扩展文档

Visual Studio Code 拥有丰富的扩展生态系统，支持各种专业化工作流。本章节涵盖关键扩展领域的文档。

## 容器工具

使用 [Docker 扩展](/docs/containers/overview) 来构建、管理和部署容器化应用。快速开始使用容器开发 [Node.js](/docs/containers/quickstart-node)、[Python](/docs/containers/quickstart-python) 或 [ASP.NET Core](/docs/containers/quickstart-aspnet-core)，并使用 [Docker Compose](/docs/containers/docker-compose) 和 [Kubernetes](/docs/containers/bridge-to-kubernetes)。

## 数据科学

直接在 VS Code 中使用 [Jupyter Notebooks](/docs/datascience/jupyter-notebooks)，通过 [Data Wrangler](/docs/datascience/data-wrangler) 探索数据，并使用 [PyTorch](/docs/datascience/pytorch-support) 和 [Azure Machine Learning](/docs/datascience/azure-machine-learning) 构建机器学习模型。

## 智能应用

使用 [Foundry Toolkit](/docs/intelligentapps/overview) 构建 AI 驱动的应用程序。创建 [智能代理](/docs/intelligentapps/create-agents)，使用 [模型](/docs/intelligentapps/models)，使用 [Playground](/docs/intelligentapps/playground)，并通过 [Agent Inspector](/docs/intelligentapps/agent-inspector) 检查智能代理行为。

## Azure

使用 [Azure 扩展](/docs/azure/overview) 部署和管理云资源。使用 [资源视图](/docs/azure/resourcesextension)，将应用部署到 [Azure App Service](/docs/azure/deployment)，并使用 [Azure Kubernetes Service](/docs/azure/aksextensions) 和 [MongoDB](/docs/azure/mongodb)。

## 远程

通过 [远程开发](/docs/remote/remote-overview) 在远程机器、容器和 WSL 上进行开发。通过 [SSH](/docs/remote/ssh) 连接，使用 [GitHub Codespaces](/docs/remote/codespaces)，或使用 [VS Code Server](/docs/remote/vscode-server) 和 [Tunnels](/docs/remote/tunnels)。

## 开发容器

使用 [Dev Containers](/docs/devcontainers/containers) 定义一致的开发环境。按照 [教程](/docs/devcontainers/tutorial) 开始入门，[创建自定义开发容器](/docs/devcontainers/create-dev-container)，并使用 [devcontainer.json](/docs/devcontainers/devcontainerjson-reference) 配置它们。
