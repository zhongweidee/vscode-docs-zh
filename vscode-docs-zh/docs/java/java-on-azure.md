# 部署 Java Web 应用

[Azure Tools](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-node-azure-pack) 扩展包包含一组丰富的扩展，使你可以轻松地发现支持 Java 应用程序的云服务并与之交互。

该扩展包支持以下开发工作流：

* 将 Java 应用程序（包括容器）部署到 [Azure App Service](https://azure.microsoft.com/services/app-service)。
* 将 Spring 微服务部署到 [Azure Spring Cloud](https://azure.microsoft.com/services/spring-cloud/)。
* 将无服务器代码部署到 [Azure Functions](https://azure.microsoft.com/services/functions)。

![Azure Tools extension](images/azure/azure-tools.png)

如果你对特定的 Azure 服务感兴趣，也可以直接在 [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/VSCode) 中搜索，查看是否有可用的扩展。
