# 传统 Java 应用程序的现代化改造

[GitHub Copilot 应用现代化扩展](https://marketplace.visualstudio.com/items?itemName=vscjava.migrate-java-to-azure)提供了强大的功能，帮助开发人员以更快的速度、更高的准确性和效率对传统 Java 应用程序进行现代化改造。

无论是升级框架、解决过时的依赖项，还是为云迁移做准备，GitHub Copilot 都利用智能自动化和与 Microsoft Azure 的深度集成来加速现代化进程，将时间从数月缩短到数天。

![GitHub Copilot App Modernization](images/java-app-mod/appmodlogo.png)

## 扩展功能

[GitHub Copilot 应用现代化](https://marketplace.visualstudio.com/items?itemName=vscjava.migrate-java-to-azure)扩展提供以下关键功能：

* **智能评估和升级规划**：Copilot 扫描您的代码库，分析依赖项和框架，并构建可操作的计划来指导您的现代化改造之旅。

* **AI 驱动的代码转换**：从 Java 8 到最新的 LTS 版本，Copilot 通过结构化的、上下文感知的编辑来自动化升级，减少重复性工作并降低升级风险。

* **可复用的迁移模式**：将您团队的重构逻辑捕获为自定义公式，然后在各个应用程序之间重新应用它们，以实现一致且可扩展的现代化改造。

* **构建和测试验证**：Copilot 修复构建错误，运行单元测试，并确保您的应用程序在更改后保持稳定，帮助维护干净的 CI/CD 管道。

* **CVE 检测和修复**：自动检测并修补升级后的已知漏洞。Copilot 应用安全修复，同时让您完全掌控更改。

* **无缝 Azure 集成**：以最小的努力将升级后的应用程序部署到 Azure。现代化后的应用程序可以立即使用 Azure AI Foundry 解锁超过 11,000 个模型、内置的可观测性、安全工具和代理服务。
