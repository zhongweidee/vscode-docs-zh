# 将应用程序部署到 Azure

Visual Studio Code 让你可以轻松地将应用程序部署到 Azure 云，我们还创建了演练指南来帮助你入门。

无论你是全栈开发人员、后端开发人员、API 开发人员还是 DevOps 工程师，都可以找到所需的部署步骤。

## 部署教程

以下教程和主题介绍了通过 Visual Studio Code 创建应用并将其部署到 Azure 的不同方式：

教程 | 描述 | 框架/语言
--- | --- | ---
[部署静态网站](https://learn.microsoft.com/azure/static-web-apps/getting-started) | 创建、部署和更新静态网站 | Angular、Blazor、React 或 Vue
[部署 Node.js 应用](/docs/nodejs/nodejs-deployment.md) | 部署 Web 应用、容器化应用或无服务器代码 | Node.js
[部署 Python 应用](/docs/python/python-on-azure.md) | 部署 Web 应用、容器化应用或无服务器代码 | Python
[部署 Java 应用](/docs/java/java-on-azure.md) | 部署 Web 应用、Spring Boot 应用或无服务器代码 | Java

你可以在 [Azure 开发者中心](https://learn.microsoft.com/azure/developer)上找到更多教程和演练指南，包括针对 JavaScript 与 Node.js、Python、Java 和 .NET 的特定语言文章。

## 后续步骤

* [部署到 Azure 容器应用](https://learn.microsoft.com/en-us/azure/container-apps/deploy-visual-studio-code) - 在无服务器平台上运行微服务和容器化应用程序。
* [Visual Studio Code Azure 扩展](/docs/azure/overview.md) - Azure 工具扩展包旨在让你在几分钟内将应用程序部署到 Azure。
* [使用 Docker](/docs/azure/docker.md) - 将你的应用程序放入 Docker 容器中，以便于重复使用和部署。
