# 我们为什么开发 Visual Studio Code？

Visual Studio Code 将源代码编辑器的简洁性与强大的开发者工具（如 IntelliSense 代码补全和调试功能）融为一体。

首先，它是一个不会妨碍您工作的编辑器。令人愉悦的低摩擦编辑-构建-调试周期意味着您可以花更少的时间在环境配置上，而将更多的时间用于实现您的创意。

## 适用于 macOS、Linux 和 Windows

Visual Studio Code 支持 macOS、Linux 和 Windows——无论您使用哪个平台，都能立即上手。

![Visual Studio Code runs on macOS, Linux and Windows](images/whyvscode/macwinlinux2.png)

## 轻松编辑、构建和调试

就其核心而言，Visual Studio Code 拥有一个极速的源代码编辑器，非常适合日常使用。凭借对数百种语言的支持，VS Code 通过语法高亮、括号匹配、自动缩进、框选、代码片段等功能帮助您立即提高生产力。直观的键盘快捷键、轻松的定制化以及社区贡献的快捷键映射让您能够轻松浏览代码。

对于严肃的编码工作，您通常需要比纯文本块更能理解代码含义的工具。Visual Studio Code 内置支持 IntelliSense 代码补全、丰富的语义代码理解和导航以及代码重构功能。

当编码变得困难时，强者会寻求调试。在轻量级的编码体验中，调试往往是开发者最怀念的功能，所以我们实现了它。Visual Studio Code 包含一个交互式调试器，让您可以逐步执行源代码、检查变量、查看调用栈并在控制台中执行命令。

VS Code 还与构建和脚本工具集成以执行常见任务，从而加快了日常工作流程。VS Code 支持 Git，因此您无需离开编辑器即可进行源代码控制，包括查看待处理的更改差异。

## 打造属于您自己的工具

您可以按自己的喜好定制每一项功能，并安装任意数量的第三方扩展。虽然大多数场景可以"开箱即用"，无需任何配置，但 VS Code 也会随着您一起成长，我们鼓励您优化体验以满足您的独特需求。VS Code 是一个开源项目，因此您也可以为 GitHub 上不断壮大且充满活力的社区做出贡献。

## 为 Web 而生

VS Code 包含丰富的内置支持，用于 Node.js 开发（包括 JavaScript 和 TypeScript），其底层技术与 Visual Studio 所采用的技术相同。VS Code 还提供了适用于 JSX/React、HTML、CSS、SCSS、Less 和 JSON 等 Web 技术的出色工具。

## 强大且可扩展的架构

在架构上，Visual Studio Code 融合了 Web 技术、原生技术和语言特定技术的优点。通过使用 [Electron](https://github.com/electron/electron)，VS Code 将 JavaScript 和 Node.js 等 Web 技术与原生应用的速度和灵活性结合在一起。VS Code 使用了一个更新、更快的版本，该版本基于与驱动"Monaco"云端编辑器、Internet Explorer 的 F12 工具及其他项目的同款工业级 HTML 编辑器。此外，VS Code 采用工具服务架构，使其能够与驱动 Visual Studio 的许多相同技术进行集成，包括用于 .NET 的 Roslyn、TypeScript、Visual Studio 调试引擎等。

Visual Studio Code 包含一个公开的扩展性模型，让开发者能够构建和使用扩展，并丰富地定制他们的编辑-构建-调试体验。

## 准备好，开始编码！

如果您更倾向于以代码编辑器为中心的开发工具，或者正在构建跨平台的 Web 和云应用程序，我们诚邀您试用 Visual Studio Code 并告诉我们您的想法！
