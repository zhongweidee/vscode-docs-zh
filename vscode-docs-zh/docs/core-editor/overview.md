# 核心编辑器功能

Visual Studio Code 内置了丰富的功能，可用于高效的代码编辑、调试和源代码控制。本节涵盖了开箱即用的核心功能。

## 配置

自定义 VS Code 以匹配您的工作流程。设置[键盘快捷键](/docs/configure/keybindings)、调整[设置](/docs/configure/settings)、管理[扩展](/docs/configure/extensions/extension-marketplace)，并为不同的项目创建[配置文件](/docs/configure/profiles)。

## 编辑代码

充分利用强大的编辑功能，例如用于智能补全的 [IntelliSense](/docs/editing/intellisense)、用于在代码库中快速浏览的[代码导航](/docs/editing/editingevolved)、[重构](/docs/editing/refactoring)工具以及[代码片段](/docs/editing/userdefinedsnippets)。

## 构建、调试和测试

运行[任务](/docs/debugtest/tasks)以自动化构建流程，使用支持多种语言的内置[调试器](/docs/debugtest/debugging)，并直接从编辑器运行您的[测试](/docs/debugtest/testing)。

## 源代码控制

VS Code 集成了 [Git 支持](/docs/sourcecontrol/overview)，具备[暂存与提交](/docs/sourcecontrol/staging-commits)、[分支管理](/docs/sourcecontrol/branches-worktrees)、[解决合并冲突](/docs/sourcecontrol/merge-conflicts)以及在 [GitHub 上协作](/docs/sourcecontrol/github)等功能。

## 终端

使用[集成终端](/docs/terminal/basics)无需离开编辑器即可运行命令。配置[终端配置文件](/docs/terminal/profiles)、利用[Shell 集成](/docs/terminal/shell-integration)并自定义[终端外观](/docs/terminal/appearance)。
