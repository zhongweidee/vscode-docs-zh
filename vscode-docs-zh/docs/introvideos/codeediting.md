# Visual Studio Code 中的代码编辑

在本 Visual Studio Code 教程中，我们将介绍如何编辑和运行一小段代码，包括下面概述的功能。观看本概述后，请继续阅读[相关资源](/docs/introvideos/codeediting.md#related-resources)部分以查看更多功能。

## 视频提纲

* 打开一个文件夹。
  * **文件** > **打开文件夹** (`kb(workbench.action.files.openFolder)`)
* 使用文件资源管理器查看文件夹中的文件和子文件夹。
  * **查看** > **资源管理器** (`kb(workbench.view.explorer)`)
* 安装 Node.js 运行时以执行 JavaScript 代码。
  * 在 [https://nodejs.org](https://nodejs.org) 上找到适合您平台的 Node.js
* 检查您的 Node.js 安装。
  * 在终端或命令提示符中，输入 `node --version`
* 创建新文件。
  * **文件** > **新建文件** (`kb(workbench.action.files.newUntitledFile)`)
* 创建一个名为 `app.js` 的简单"Hello world"控制台应用程序。
* IntelliSense 在您输入时提供建议。
* 自动格式化源代码。
  * **格式化文档** 命令 (`kb(editor.action.formatDocument)`)
* 开启自动保存。
  * **文件** > **自动保存**
* 显示集成终端。
  * **查看** > **终端** (`kb(workbench.action.terminal.toggleTerminal)`)
* 拆分终端。
  * **拆分终端** (`kb(workbench.action.terminal.split)`)
* 创建新终端。
  * **创建新终端** (`kb(workbench.action.terminal.new)`)
* 运行应用程序。
  * 在集成终端中，输入 `node app.js`
