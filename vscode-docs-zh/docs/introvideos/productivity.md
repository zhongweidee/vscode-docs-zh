# 生产力技巧

在本 Visual Studio Code 教程中，我们将介绍有助于提高编码生产力的 Visual Studio Code 功能。学习一些技巧即可加快工作流程，帮助你发掘 VS Code 编辑器的全部潜力。

## 视频大纲

* 双击 Shift

  * 在键盘快捷方式编辑器中为常用命令分配双击 `kbstyle(Shift)` 键盘快捷键（**Preferences: Open Keyboard Shortcuts** 命令）

* 使用键盘快捷键聚焦侧边栏

  * 按下 `kb(workbench.action.focusSideBar)` 将焦点置于主侧边栏
  * 按下 `kb(workbench.action.focusFirstEditorGroup)` 将焦点移回编辑器

* 复制粘贴

  * 按下 `kb(editor.action.clipboardCopyAction)` 复制整行
  * 按下 `kb(editor.action.clipboardPasteAction)` 粘贴整行

* 多重光标

  * 按下 `kb(editor.action.insertCursorAtEndOfEachLineSelected)` 在每行选中内容的末尾添加光标

* 多个终端窗口

  * 在编辑器中拖动终端窗口
  * 使用 `kb(workbench.action.quickOpen)` 在编辑器中切换打开的终端
