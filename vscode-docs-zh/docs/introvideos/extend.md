# 在 Visual Studio Code 中使用扩展

使用 Visual Studio Code 扩展来添加新功能、主题等。在本教程中，我们将向你展示如何查找扩展、安装你喜欢的扩展，以及禁用你不想一直使用的扩展。

## 视频大纲

* 使用扩展视图查找要安装的扩展。
* 从 VS Code 扩展市场安装扩展。
* 通过**功能贡献**选项卡或命令面板（`kb(workbench.action.showCommands)`）查看添加了哪些功能。
* 查看其他扩展的推荐。
