# Visual Studio Code 中的调试

调试是 Visual Studio Code 的核心功能。在本教程中，我们将向你展示如何在 VS Code 中运行和调试程序。我们将浏览**“运行和调试”**视图，探索一些调试功能，最后设置一个断点。

> **提示：** 要使用本视频中演示的 Node.js 调试功能，你需要先安装 [Node.js](https://nodejs.org)。要跟随视频中的 Python 部分学习，你需要安装 [Python](https://www.python.org/downloads/)。
