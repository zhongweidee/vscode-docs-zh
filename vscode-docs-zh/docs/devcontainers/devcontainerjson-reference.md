# devcontainer.json 参考

## 开发容器规范

正如[开发容器主文档](/docs/devcontainers/containers.md)中所述，我们正在制定**开发容器规范**，以使任何人都能在任何工具中配置一致的开发环境。

`devcontainer.json` 参考[托管在规范网站上](https://containers.dev/implementors/json_reference)。您也可以在此处查看 [devcontainer.json 架构](https://containers.dev/implementors/json_schema)。

您还可以在 [devcontainers/spec](https://github.com/devcontainers/spec/tree/main/docs/specs) 仓库中查看规范文档和最新提案。

## 其他资源

[创建开发容器](/docs/devcontainers/create-dev-container.md)提供了有关配置开发容器的更多信息，或者您可以使用命令面板 (`kbstyle(F1)`) 中的 **Dev Containers: Add Dev Container Configuration Files...**（开发容器：添加开发容器配置文件...）或 **Codespaces: Add Dev Container Configuration Files...**（Codespaces：添加开发容器配置文件...）命令来添加各种基础配置。

> **提示：** 如果您已经构建了一个容器并连接到它，请务必从命令面板 (`kbstyle(F1)`) 中运行 **Dev Containers: Rebuild Container**（开发容器：重建容器）或 **Codespaces: Rebuild Container**（Codespaces：重建容器）来应用您所做的任何更改。
